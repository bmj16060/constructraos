# Architecture Decision Records

This folder holds Architecture Decision Records for ConstructraOS.

Conventions:

- one file per decision
- filenames use `NNNN-short-title.md`
- statuses should be one of `proposed`, `accepted`, `superseded`, or `deprecated`
- ADRs should stay narrow and record the decision, the reasoning, and the consequences
